HTML

HTML er en forkortelse for HyperText Transfer Language

HTML er sproget man benytter til at lave hjemmesider som kan vises i en webbrowser.

Man kan tænke på HTML som en websides krop. Det bygges op med head og body.

I head skriver man sidens meta tags, hvor man kan pege på CSS og Javascript filer, som HTML dokumentet skal drage ressourcer fra


CSS

CSS er en forkortelse for Cascading Style Sheet

Hvis man tænker på HTML som websidens krop, så er CSS tøjet

CSS er altså en websides styling - ved hjælp af CSS kan man style placering, tekststørrelser, type, farve samt bagrundsfarver.

Man kan enten bruge CSS direkte på HTML tags, eller man kan have det i en sepereret fil, som HTML filen peger på, ved at indsætte CSS filens link i HTML'ens head afsnit
