API

API er en forkortelse for Apllication Programming Interafce:
API er et computersprog der gør at to maskiner kan kommunikere sammen ved brug af json format.

HTTP requests:

Der er flere forskellige HTTP request, herunder er de mest populære oplistet:

  • POST - skriv i database
    •
  • GET - læs fra databasen
    •
  • PUT - opdaterer i databasen eller opret ny hvis ikke eksisterende
    •
  • PATCH - opdaterer
    •
  • DELETE - sletter fra databasen
    •

    JSON

    JSON står for JavaScript Object Notation, open data format som er læseligt for mennesker og kan transmiterer data objekter mellem systemer.
    Det bruges i stedet for XML og AJAX.
